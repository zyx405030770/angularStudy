1.angular环境搭建 创建项目 运行项目

  • 以下块为news组件

    1. 环境搭建

      1. 安装node.js:
        安装node.js稳定版
      2. 安装cnpm:
        npm install -g cnpm --registry=https://registry.npm.taobao.org
      3. 安装angular脚手架:
        npm install -g @angular/cli
        cnpm install -g @angular/cli
      4. 查看版本
        angular：ng v
        node: node -v
        npm: npm -v
        cnpm: cnpm -v

    2. 创建项目

      1. ng new [项目名称] -----------------# 创建项目
      2. ng new [项目名称] --skip-install --# 不含装第三方依赖的情况下创建项目
      3. yarn install ------------------------# 单独安装第三方模块包

    3. 运行项目

      1. cd [项目名称] # 跳转到项目下
      2. ng serve --open or ng serve -o # 运行项目
      3. ng serve --open --port [端口号] # 指定端口运行项目
